For graders: Even if you're familiar with Pokemon, you may find it useful to read the following rundown listing functionality you may otherwise miss.

  • Invalid input will result in an alert asking to enter a valid name or number
  • Valid Pokemon numbers are between 1 and 807
  • Some common Pokemon names to test could be "Pikachu", "Lucario", "Absol" or any others you may know of
  • The list of Pokemon flows. If you enter Pokemon #001 or #807 then the "previous" and "next" Pokemon will the #807 and #001 respectively
  • The little image in the box alternates between front and back if the API allows it. Otherwise the image will only be the front view. This can be tested with #1, which has front and back view, or #807, which only has front view
  • There'll be a loading image in place of the little Pokemon image while it loads
  • There'll be a button to take you back to this instruction menu
